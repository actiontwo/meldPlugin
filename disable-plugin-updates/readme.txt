== Description ==

A simple WordPress plugin that disables all the updating of plugins.

This plugin is no longer under development because it has been combined with: <a href="http://wordpress.org/plugins/stops-core-theme-and-plugin-updates">Disable Updates Manager</a>



= Please note!!! =


It is important to keep your plugins up to date. 
If you use this plugin, make sure your plugins is not harming your website.

== Installation ==


1. Go to the Plugins> Add New> Upload in you admin dashboard.

2. Upload the zip file and install.
3. Go to the Plugins section in your admin dashboard and activate.
